What is HTML?

HTML, which stands for HyperText Markup Language, is the standard language used to create the structure of webpages. It organizes content such as headings, paragraphs, images, links, and lists so that web browsers can display them correctly.

Although HTML does not control the appearance of a webpage, it provides the foundation that other technologies, such as CSS and JavaScript, build upon. Every webpage on the internet uses HTML in some form.

Common HTML Tags

<h1>

Headings

Creates the main heading or title of a webpage.

<p>

Paragraphs

Used to organize text into readable paragraphs.

<a>

Links

Creates hyperlinks that connect webpages together.

<img>

Images

Displays pictures and graphics on a webpage.

Basic HTML Structure

Just like a house needs a strong foundation, every webpage begins with a basic HTML structure. This structure helps organize the webpage into different sections so that browsers know how to display text, images, and other content. Understanding this layout is one of the first steps in learning web development. 

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>My Webpage</title>
    </head>

    <body>
        <h1>Hello World!</h1>
        <p>This is my first webpage.</p>
    </body>
</html>
