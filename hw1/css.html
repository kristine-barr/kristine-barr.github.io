What is CSS?

CSS, which stands for Cascading Style Sheets, is the language used to control the appearance of a webpage. While HTML creates the structure and content, CSS adds colors, fonts, spacing, layouts, and other visual styles that make websites more attractive and easier to use.

By separating the design from the content, CSS allows developers to create consistent styles across multiple webpages. This makes websites easier to maintain and provides visitors with a better browsing experience.

Common CSS Properties

color

Changes the color of text.

color: navy;

background-color

Changes the background color of an element.

background-color: yellow;

font-size

Changes the size of text.

font-size: 24px;

padding

Adds space inside an element.

padding: 20px;
